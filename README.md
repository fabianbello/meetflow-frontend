# D-Meetflow-frontend
Tech demo D-Meetflow creada utilizando framework D-Meetflow-backend https://github.com/fabianbello/meetflow-backend

# Requisitos:
1. Node v20
2. Angular js v16.2.5

# Instalación local:

# 1. Instalar e incializar D-Meetflow-backend https://github.com/fabianbello/meetflow-backend
# 2. Inicializar aplicación:
En terminal:
1. npm install --force
2. ng serve
# 3. Ingresar a la plataforma web
En navegador:
1. ingresar al enlace: "http://localhost:4200/"
